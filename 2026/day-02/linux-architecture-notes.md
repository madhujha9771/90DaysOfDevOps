# Linux Architecture
Linux Architecture is defined in such way that how different components of the system interact with each other to manage hardware resources,application,shell and utilities.

## The Main Components of Linux operating system

-Hardware 
-Kernel
-shell
-Application

# Harware

-It is the physical component which handles the functioning of the system efficently and relibly ex-RAM,CPU,ROM etc.

# Kernel

-It sits between Hardware and userspace that handles resource allocation ensuring smooth communication between software and hardware.


# Shell

-Users Communicate with Linux operating system using shell,it interprets and executes command entered in a terminal, Bash,Zsh,Fish etc.

# Application
-These are the Software programs that you as the user interact with directly.
